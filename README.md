# Cyberbullying_Detection_FYP1

## Project Overview
This repository contains the code and notebooks for my Final Year Project (FYP 1). The primary goal of this project is to develop and compare deep learning models for detecting multilangual cyberbullying. 

## Dataset 
* **Dataset Status:** The dataset used for this training is completely balanced.

## Models Implemented
This project experiments with and compares the following architectures:
1. **CNN (Convolutional Neural Network)**
2. **BiLSTM (Bidirectional LSTM)**
3. **CNN + BiLSTM**
4. **XLM-RoBERTa**

## Current Progress & Observations
* **Current Status:** Phase 1 training and evaluation.
* **Model Behavior:** During initial evaluations, it was observed that the model is heavily recalling the non-hate class. Hyperparameter tuning and further analysis are ongoing.

## How to Run the Code
you can clone this repository and run the `.ipynb` files locally using Jupyter Notebook.
